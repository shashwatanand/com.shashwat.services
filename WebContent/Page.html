Since javascript runs on the UI thread this means that the rest of the browser can't be interacted with until the response is received.

Asynchronous Calls

These calls will return immediately after the request is sent.

This means that any information that the asynchronous call receives cannot be returned in the function it was called in.

However this no longer locks the UI.
